Extend any clip naturally

Continue the motion, loop for seamless ambience, or stretch a hero shot. The Video Extender, used right.

Intermediate 4 min read Updated 2026-05-25

Video Extend predicts the next seconds of any clip — like an autocomplete for motion. It's the simplest fix when a generation ends just a little too soon.

Plain continuation

Leave the prompt blank and the model continues the existing motion. Useful when a 4-second clip needs to be 6 to fit your edit.

Directed continuation

Add a prompt and the model steers the extension toward it. Example: a clip of a person walking, with the prompt “turns and looks at the camera, smiles” — Lumen will continue the walk and add the turn.

Loop mode

When loop is on, the model engineers the last frame to match the first frame of the source. The result loops seamlessly — perfect for ambient backgrounds and scroll-stoppers.

Extra seconds

2-4 seconds extend cleanly. 6+ seconds risks the model drifting. For anything longer than 4, use Storyboard instead.
